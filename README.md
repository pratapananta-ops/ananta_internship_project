# Phishing Awareness Simulation Using Social Engineering Techniques

## Project Overview
This project demonstrates how phishing attacks use social engineering techniques to trick users into revealing sensitive information. A simulated phishing email and fake login page were created to analyze user responses and raise awareness about cybersecurity threats.

## Objectives
- Understand phishing attacks and social engineering techniques
- Design a phishing email simulation
- Create a fake login page using HTML
- Analyze user responses to phishing attempts
- Suggest preventive measures against phishing attacks

## Tools & Technologies Used
- HTML
- Netlify (for hosting the webpage)
- Google Forms (for collecting responses)
- Microsoft Excel (for analyzing results)

## Methodology
1. Studied phishing attack techniques.
2. Designed a phishing email resembling a security alert.
3. Created a fake login webpage using HTML.
4. Hosted the webpage using Netlify.
5. Shared the phishing email for experimental purposes.
6. Collected responses using Google Forms.
7. Analyzed the results using charts and graphs.

## Results
The experiment showed that some participants interacted with the phishing email link, demonstrating how easily users can be targeted by social engineering attacks.

## Preventive Measures
- Verify email sender addresses
- Avoid clicking suspicious links
- Use multi-factor authentication
- Conduct cybersecurity awareness training
